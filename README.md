> ⚠️ **First-time setup:** Go to **Settings → Pages → Build and deployment → Source** and select **GitHub Actions**, then re-run the deploy workflow (or push to `main`) to publish the site.

## Development

```bash
cd app
npm install
npm run dev
```

## Build

```bash
cd app
npm run build
```

The production build output is in `app/dist/`.

## Deployment

Pushes to `main` automatically trigger the GitHub Actions workflow (`.github/workflows/deploy.yml`) which builds and deploys the site to GitHub Pages.
